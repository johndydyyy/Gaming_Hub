#  GamingHub

A premium e-commerce platform dedicated to high-performance gaming PCs and components. Designed with a sleek aesthetic for enthusiasts to browse, customize, and manage their gaming gear.

---

##  Features

* **Premium Dark UI:** A modern, cohesive design optimized for the gaming community.
* **Featured Builds:** High-end rigs (Pulsar, Nova, Aurora) with detailed specs for CPU, GPU, and Cooling.
* **Interactive Shopping Cart:** Live quantity updates, subtotal calculations, and item removal.
* **Integrated Wallet:** A digital balance system allowing users to add funds and track transaction history.
* **Secure Account Management:** Centralized dashboard for managing profile info, contact details, and payment methods.

##  Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript 
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** XAMPP / Apache

---

##  Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone [https://github.com/johndydyyy/GamingHub.git](https://github.com/johndydyyy/GamingHub.git)
   
2. **Move to Web Directory Copy the folder to your XAMPP htdocs directory**

3. **Database Setup**

* **Open phpMyAdmin.

* **Create a new database named gaming_hub.

* **Import the database.sql file provided in the repository.

4. **Run the Project**

* **Start Apache and MySQL in XAMPP.

* **Open your browser and go to localhost/GamingHub/login.php.
